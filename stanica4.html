Ќе се запознаеш со условни структури во Python


Ќе креираш синџири на повеќе услови


Ќе се запознаеш со вгнездената if-else изјава

Условните структури се користат за проверка на одредени услови и извршување на соодветни делови од кодот во зависност од резултатот на условот. Условни структури се: if-else, if-elif-else.

Синтакса на if исказот




if condition:

statement 1
statement 2
statement n

Синтакса на if-else исказот




if condition:

statement 1

else:

statement 2

Да го видиме примерот на изјавата if-else.

Во овој пример, ќе внесеме пасворд и ќе провериме дали е точниот или не.
from js import document def runTextInTag(id): src = document.getElementById(id).value exec(src)





Синтакса на if-elif-else исказот




if condition-1:

statement 1

elif condition-2:

statement 2

elif condition-3:

statement 3

...

else:

statement

Да го видиме примерот на изјавата if-else.

Во овој пример, ќе споредиме двете бројчиња, кој е поголем или помал или и двете бројчиња се еднакви
from js import document def runTextInTag(id): src = document.getElementById(id).value exec(src)
