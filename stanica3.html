Оператори

На третата станица од твоето патување во

светот на Python:


Ќе ги разбереш операторите за доделување


Ќе се запознаеш со аритметички, логички, споредбени и други видови на оператори

Операторите се користат за извршување на операции на променливи и вредности.

Aритметички оператори

Аритметичките оператори се користат со нумерички вредности за извршување на заеднички математички операции:

# Оператори Имиња Примери
1 + Собирање x+y
2 - Одземање x-y
3 * Множење x*y
4 / Делење x/y
5 % Апсолутна вредност на реален или комплексен број x%y
6 ** Експоненцијација x**y
7 // Поделбата на подот x // y



Оператори за доделување

Операторите за доделување се користат за доделување вредности на променливите:

# Оператор Пример Исто како
1 = x = 5 x = 5
2 += x += 3 x = x + 3
3 -= x -= 5 x = x - 5
4 /= x /= 3 x = x / 3
5 *= x *= 5 x = x * 5
6 %= x %= 3 x = x % 3
7 //= x //= 5 x = x // 5
8 **= x **= 3 x = x ** 3
9 &= x &= 5 x = x & 5
10 |= x |= 3 x = x | 3
11 ^= x ^= 5 x = x ^ 5
12 >>= x >>= 3 x = x >> 3
1 <<= x <<= 5 x = x << 5



Логички оператори

Логичките оператори се користат за комбинирање на условни искази:

# Оператори Опис Примери
1 and Враќа True ако двете изјави се вистинити x < 5 and x < 10
2 or Враќа True ако едно од тврдењата е точно x < 8 or x < 4
3 not Превртете го резултатот, враќате Неточно ако резултатот е вистинит not(x < 5 and x < 10)



Оператори за идентитет

Идентитетските оператори се користат за споредување на објектите, не ако се еднакви, туку ако се всушност истиот објект, со иста мемориска локација:

# Оператори Опис Примери
1 is Враќа True ако двете променливи се ист објект x is
2 is not Враќа True ако двете променливи не се ист објект x is not



Bitwise оператори

Bitwise операторите се користат за споредување (бинарни) броеви:

Оператори Имиња Опис Примери
& AND AND x & y
| OR AND x | y
^ XOR AND x ^ y
~ NOT AND ~ x
<< ve AND x << 2
>> Експоненцијација x**y AND
% AND Поделбата на подот x%y
