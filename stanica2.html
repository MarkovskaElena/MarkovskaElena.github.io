Откако инсталираме соодветна развојна околина за Python, веќе ќе научиме што се променливи како и каде се користат како и за влезно/излезните функции во Python. Пред да почнеме со пишување код да се запознаеме со неколку нови поими кои понатаму ќе ги употребуваме:

1. Варијабли/Променливи

Променливи или варијабли, се посебни локации во меморијата на компјутерот кои ги користиме за да чуваме вредност. Постојат правила при пишување на имиња на променливи во Python:

- Името на променливата мора да започне со буква или знак на долна црта.
- Името на променливата не може да започне со број.
- Името на променливата може да содржи само алфа-нумерички знаци и долни црти (Az, 0-9 и _ ).
- Името на променливата не може да биде ниту еден од клучните зборови на Python.
- Имињата на променливите се чувствителни на големи букви (age, Age и AGE се три различни
променливи).

Правилно се именуваат променливи:
myname = “Elena”
my_name = “Elena”
_my_name = “Elena”
myName = “Elena”
MYNAME = “Elena”
myname9 = “Elena”
Неправилно се именуваат променливи:
1myname = “Elena”
my-name = “Elena”
my name = “Elena”



2. Податочни типови

Секоја променлива има свој податочен тип. Променливите во Python не мора да се декларираат сo точно одреден тип. Во моментот кога доделуваме вредност на променливата, Python автоматски го одредува типот на податокот кој го прима.

x = int(20)


Mоже да има целобројна вредност

x = float(20.5)


Mоже да има децимална вредност

x = bool(5)


Mоже да има булова вредност

x = range(6)


Mоже да има ранг

x = str("Hello World")


Mоже да има текст

x = list ["apple", "banana", "cherry"]


Mоже да има листи

x = tuple("apple", "banana", "cherry")


Mоже да има торки

4. Промена на податочен тип

Casting во програмирањето се однесува на промена на типот од еден во друг податочен тип. Треба да се внимава при користење на casting бидејќи може да дојде до губење на информациите или да дојде до неправилни резултати. На пример: кога претвораме децимален број во цел број го губиме децималниот дел.

int() - конструира цел број од цел број буквално, float literal (со отстранување на сите децимали) или стринг буквално (под услов низата да претставува цел број)
from js import document def runTextInTag(id): src = document.getElementById(id).value exec(src)
float() - конструира пловечки број од цел број буквално, float literal или стринг буквално (под услов низата да претставува float или цел број)
from js import document def runTextInTag(id): src = document.getElementById(id).value exec(src)
str() - конструира низа од широк спектар на типови на податоци, вклучувајќи низи, целобројни литерали и float литерали
from js import document def runTextInTag(id): src = document.getElementById(id).value exec(src)
